## 🚀 Overview

**Year 2157.** Earth’s economy has collapsed.  

You are a broke interstellar merchant with:
- 🚀 One damaged ship  
- 💰 500 Galactic Credits  
- 🧠 Zero financial knowledge  

Your mission:  
**Build wealth across the galaxy and achieve Financial Freedom.**

Players think they’re trading commodities.  
They’re actually learning how money works.

## 🎮 Core Experience

**Every action is a financial decision. Every decision has consequences.**

### 📈 Trade & Grow
- Buy low on one planet, sell high on another  
- Real-time price fluctuations  
- Learn arbitrage and market timing  

---

### 🚨 Survive Market Crashes
- Random galactic events trigger volatility  
- 30-second decision window  
- Choose:
  - Panic Sell  
  - Hold  
  - Buy the Dip  
  - Move to Safe Haven  

Experience real investor psychology under pressure.

---

### 🚀 Upgrade Your Ship (Your Financial Health)

Your ship evolves based on how you manage money:

- ⚡ Engine → Income generation  
- 📦 Cargo → Investment capacity  
- 🛡 Shields → Emergency fund  
- 🧭 Navigation → Financial knowledge  

---

### 📊 Track Your Portfolio
- Real-time net worth  
- Profit & loss tracking  
- Asset allocation visualization  
- Diversification score  

---

### 🧠 Learn Without Realizing

Every mechanic maps to real finance:

- Trading → Arbitrage  
- Crashes → Market volatility

- Safe assets → Risk management  

You don’t read finance. You live it.

## 🛠️ Tech Stack

Built entirely with core web technologies:

- HTML5  
- CSS3 (animations + dark UI)  
- Vanilla JavaScript  
- Canvas API (galaxy map & charts)  
- localStorage (persistent game state)  

No frameworks. No dependencies. Fully offline-capable.
